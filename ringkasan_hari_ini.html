Apa yang Kami Kerjakan Hari Ini

Tim kreatif memberikan masukan di kolom Feedback di 5 tab bible (Characters, Locations, Costume, Props, Effects). Kami menerapkan semua perubahan, membuat ulang gambar referensi yang terdampak, lalu menandai sel yang sudah dikerjakan dengan warna hijau muda — supaya tim bisa langsung melihat apa yang sudah ditindaklanjuti.

Selain itu, struktur Google Sheet juga disederhanakan supaya lebih mudah dirawat ke depannya: semua "preamble" prompt sekarang hidup di satu tempat saja (bagian atas tab), dan body per-set bisa diedit langsung tanpa duplikasi.

Restrukturisasi Sheet
Format "Living Document" diterapkan
Sebelumnya: preamble seperti "Shot with Arri 35" diulang 86 kali di tab Video Prompts. Sekarang: preamble hidup di satu sel saja (B1, B2). Body shot ada di kolom yang bisa diedit. Kalau preamble berubah (misal ganti kamera), semua 17 set otomatis ikut berubah lewat formula. Single source of truth
Folder Video
17 folder per-set dibuat di Drive
Struktur: videos/set-01/, videos/set-02/, … videos/set-17/. Semua sudah di-set "anyone with link" supaya bisa dibuka tim tanpa minta akses. Kolom E di tab Video Prompts sudah otomatis diisi link folder masing-masing.
HTML Tambahan
Skrip dialog episode disusun ulang dalam urutan shot
Semua dialog dari Shotlist (kolom G) ditarik ke satu halaman HTML format screenplay — tampak seperti naskah rapi untuk aktor & sutradara. Bisa dicetak langsung untuk reading session.

Yang Sekarang Bisa Tim Lakukan

  • Buka galeri HTML sesuai peran: pharaoh_king_gallery.html (tim) · ..._CLIENT.html (client review) · ..._PRODUCTION.html (crew di lokasi)
  • Edit sekali, semua tab ikut terupdate — ubah B1 di tab Video Prompts, semua 17 set ikut berubah otomatis
  • Edit body shot di tab Storyboard Prompts kolom C → tab Video Prompts kolom C otomatis ikut update lewat formula lintas-tab
  • Lanjut ke tahap video generation — semua folder Drive sudah siap menerima file output per-set
  • Cetak skrip dialog langsung dari pharaoh_king_dialogue.html untuk reading session aktor

Catatan untuk Tim Kreatif

"Beri masukan apapun di kolom Feedback di tab bible — tidak perlu diformat khusus. Pipeline akan baca, terapkan perubahan, dan tandai hijau saat sudah selesai."
"Kalau mau preamble berbeda untuk satu set tertentu, jangan ubah B1 — buat override di body langsung. Default tetap berlaku untuk set lainnya."
"Galeri client (..._CLIENT.html) bisa langsung dikirim ke client tanpa edit lebih lanjut — sudah bersih dari catatan internal."
