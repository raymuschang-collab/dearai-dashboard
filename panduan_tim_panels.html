05 / 11 STORYBOARD

/imggen
4 MODE

Default = stick figure tanpa wajah. Untuk coverage validation, BUKAN deliverable.

DEFAULT --stick STICK FIGURES Featureless. No face. No wardrobe. Black pen on white. Untuk validasi blocking + camera + staging. Sajangnim aesthetic
--pencil PENCIL SKETCH Director-pad sketch. Features readable tapi loose. Untuk review yang butuh fidelity sedikit lebih tinggi. Mid-fidelity
--photoreal PHOTOREAL Full photoreal stills (Arri Alexa 35, Kodak 250D). MAHAL. Pakai HANYA untuk hero deliverable. Hero shots only
--sheet SHEET LEGACY Pakai apapun yang ada di B1:B4 sheet globals. Cuma kalau tahu persis kenapa. Advanced only
⚠ Aturan emas: Stakeholder masih debat likeness di stage storyboard? Bukan ship photoreal — lock cast bible dulu sebelum storyboards.

07 / 11 BULK MODE

BULK
COMMANDS

Sekali fire, semua set / semua bible jadi. Idempotent — re-run skip yang udah Done.

/imggen-all-storyboards SEMUA STORYBOARD Fire stick-figure storyboards untuk semua Pending sets. ~25 menit untuk 14-set episode. ~25 min · $1.10
/imggen-all-assets SEMUA BIBLE Generate ref images untuk CHARACTERS + LOCATIONS + PROPS + COSTUME + EFFECTS. Routing per bible. ~30 min · varies
/byteplus-upload-all UPLOAD KE BYTEPLUS Upload semua bible refs ke BytePlus. Auto-skip row yang udah Uploaded. ~10 min
/vidgen-all-sets SEMUA VIDEO Loop SP Pending rows, fire 1 video per set sequentially. Resume-friendly. ~45 min · $17 per ep

10 / 11 TIPS EMAS

ATURAN
EMAS.

✓ SEBAIKNYA
  • Mulai dari Asset Library — pastikan @-mention semua resolve dulu
  • Pakai 480p default untuk testing — 1080p mahal, simpan untuk hero
  • Test 1 set dulu sebelum fire-all — /vidgen set 1 V1 sebelum /vidgen-all-sets
  • Update Asset Library, jangan hardcode kode di prompt
✗ JANGAN
  • Delete BytePlus asset tanpa update Asset Library — vidgen error
  • Fire 1080p tanpa --confirm — mahal 2.6×, harus preview prompt dulu
  • Re-fire job gagal tanpa cek alasan — biasanya ada alasan jelas
  • Share kunci .env di chat / Discord / Slack — itu privacy leak

KENAPA SPLIT BEGINI?

Higgsfield = subscription paid, $0 marginal — pakai default semua karakter/prop/effect/storyboard. Reve = lebih bagus untuk wide environment shot, jadi lokasi tetap di sana. BytePlus = Seedance 2 punya HK wallet $1M, video gen hanya di sini.
